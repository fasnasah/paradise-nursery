# e-plantShopping

## Paradise Nursery Shopping Application

This project is a React + Redux based shopping application that allows users to browse houseplants and add them to a cart.

### Features

* Landing page with background image
* Product listing page with plant categories
* Shopping cart functionality
* Add to cart and update quantity
* Dynamic cart counter
* Checkout button (Coming Soon)

### Technologies Used

* React
* Redux Toolkit
* React Router
* CSS
